# Tau-Phah-Ji-Command
鬥拍字的指令直接包做一个套件

## Tau
```
pip install Tau-Phah-Ji-Command
```

## Ing
```
>>> from tauphahji_cmd import tàuphahjī
>>> tàuphahjī('gau5 tsa2\nlâi-khì.')
```
